C is an influential [programming language]($p/programming-language) developed in
the 1970s by Dennis Ritchie. Many newer languages are inspired by C, often
sharing similar [syntax]($s/syntax). These languages are called the *C family*
of languages.

## [Hello, world!]($h/hello-world)

```c
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
```
