C++ is a [programming language]($p/programming-language) created in 1985 by
Bjarne Stroustrup as an extension of [C]($c/c). Some notable changes compared
to C include the addition of [object-oriented programming]($o/object-oriented-programming)
patterns such as `class`, a new [standard library]($c/cpp-std), and improved
[memory management]($m/memory-management).

## [Hello, world!]($h/hello-world)

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}
```
