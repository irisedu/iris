Memory is the part of a [computer]($c/computer) which is responsible for storing
the code and state of [programs]($p/program).

It is not to be confused with *storage*. In contrast to storage, memory is
typically lost/cleared when a computer is turned off. The integrated circuits
("chips") which make up the memory are thus referred to as *volatile memory*.
